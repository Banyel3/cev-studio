# cev.studio

One-page landing site for **cev.studio**, a digital studio (web, mobile, brand, 3D). The contact form is the hero; submissions persist to Supabase and are viewable at a password-gated `/admin`.

Built with Next.js (App Router), TypeScript, Tailwind CSS, and Supabase.

> Setup, environment, and deploy instructions are added in the final commit.
